# E-187-03: Codify Season-Length Calibration in Context Layer

## Epic
[E-187: Threshold Calibration for Youth/HS Seasons](epic.md)

## Status
`TODO`

## Description
After this story is complete, the context layer will include a season-length calibration rule that fires when any agent touches display surfaces, the baseball-coach agent definition will include season-length context, and the stale CLAUDE.md threshold reference will be corrected.

## Context
The user explicitly wants the coaching-season-length philosophy baked into the context layer so every agent building display thresholds defaults to youth/HS-appropriate values. This prevents future agents from re-introducing pro-calibrated thresholds. The baseball-coach agent definition should also reflect that the program plays 25-35 game seasons -- data is sparse, show what you have.

## Acceptance Criteria
- [ ] **AC-1**: A new rule file `.claude/rules/season-length-calibration.md` exists with `paths:` frontmatter scoped to display surfaces (`src/reports/**`, `src/api/templates/**`, `src/charts/**`, `src/api/routes/dashboard.py`).
- [ ] **AC-2**: The rule contains the principle: display thresholds must be calibrated for 25-35 game seasons, not 162-game professional seasons. Show data at minimum reasonable thresholds; improve display as depth grows. Don't suppress useful data behind conservative cutoffs.
- [ ] **AC-3**: The rule includes the current threshold reference values (5 PA batting, 6 IP pitching, 3 BIP spray, 20 BIP team spray) so agents have concrete numbers, not just a philosophy.
- [ ] **AC-4**: `.claude/agents/baseball-coach.md` is updated to include season-length context in the Identity or Lincoln-Specific Context section: programs play 25-35 game seasons, data is sparse early in the season, coaches need to see trends from limited data.
- [ ] **AC-5**: The existing "20 PA or 15 IP" threshold references in `.claude/agents/baseball-coach.md` (Sample Size Awareness section ~line 69 and Anti-Patterns ~line 98) are updated to match the new thresholds (5 PA / 6 IP).
- [ ] **AC-6**: The CLAUDE.md Data Model section's spray chart display thresholds reference is updated from "10 BIP minimum for per-player charts" to "3 BIP minimum for per-player charts" (matching the actual code value).

## Technical Approach
Create a new scoped rule file. Update the baseball-coach agent definition's existing section. Fix one line in CLAUDE.md. All three changes are context-layer files, per Technical Notes TN-5.
