## Description
After this story is complete, the FastAPI application started in E-009-02 serves
server-rendered HTML dashboard pages using Jinja2 templates and Tailwind CSS CDN. The
dashboard is reachable at `http://localhost:8000/dashboard` and displays sample baseball
stats. An agent using WebFetch can fetch that URL and receive populated HTML with
recognizable stat content -- confirming agent browsability works.

## Context
E-009-02 built the FastAPI app skeleton with a health endpoint and SQLite connection. This
story adds the dashboard layer: Jinja2 templates, a base layout, a sample team stats page,
and Tailwind CDN for styling.

The coaching dashboard is deliberately simple at this stage. The goal is to prove that:
1. Server-rendered HTML works from FastAPI + Jinja2
2. The baseball-coach agent can browse a localhost dashboard URL via WebFetch
3. The template structure is set up correctly for E-004 (real dashboard views) to build on

This story does NOT implement real coaching views (that is E-004). It creates the template
scaffold with enough sample data to demonstrate agent browsability and mobile layout.

Key design decisions from the epic and decision record:
- **Single app**: Dashboard routes live in the same FastAPI process as the API. No
  separate dashboard service.
- **Server-rendered HTML**: Jinja2 templates render on the server. JavaScript is not
  required to display data. WebFetch receives fully populated HTML.
- **Tailwind CDN**: No build step for MVP. Include the CDN `<script>` tag in `base.html`.
- **Mobile-first**: Base template uses viewport meta tags and a mobile-first CSS approach.

## Technical Approach

### Jinja2 Setup in FastAPI

```python
# src/api/main.py (additions to E-009-02 base)
from fastapi.templating import Jinja2Templates
from fastapi.staticfiles import StaticFiles

templates = Jinja2Templates(directory="src/api/templates")
app.mount("/static", StaticFiles(directory="src/api/static"), name="static")
```

### base.html Sketch

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>{% block title %}Baseball Dashboard{% endblock %}</title>
  <script src="https://cdn.tailwindcss.com"></script>

<body class="bg-gray-50 text-gray-900">
  <nav class="bg-blue-900 text-white px-4 py-2">
    <span class="font-bold">LSB Baseball</span>
  </nav>
  <main class="p-4">
    {% block content %}{% endblock %}
  </main>
</body>
</html>
```

### Dashboard Route

```python
# src/api/routes/dashboard.py
from fastapi import APIRouter, Request
from starlette.concurrency import run_in_threadpool
from ..db import get_team_batting_stats

router = APIRouter()

@router.get("/dashboard")
async def team_stats(request: Request):
    players = await run_in_threadpool(get_team_batting_stats)
    return templates.TemplateResponse(
        "dashboard/team_stats.html",
        {"request": request, "players": players, "team_name": "Lincoln HS"}
    )
```
