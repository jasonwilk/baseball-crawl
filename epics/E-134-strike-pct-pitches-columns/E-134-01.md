# E-134-01: Add Strike % and # Pitches to all pitching display surfaces

## Epic
[E-134: Add Strike % and # Pitches to Pitching Displays](epic.md)

## Status
`TODO`

## Description
After this story is complete, all five pitching display surfaces in the coaching dashboard will show two new columns: **# Pitches** (total pitches thrown) and **Strike %** (total_strikes / pitches × 100). The data already exists in the database; this story adds the columns to SQL queries, computes the rate in Python, and renders them in Jinja2 templates.

## Context
Coaches want pitch efficiency data alongside existing rate stats. The `pitches` and `total_strikes` columns are already populated in `player_season_pitching` and `player_game_pitching` by all three loaders (game, season stats, scouting). This is a query + display change only.

- [ ] **AC-2**: The Opponent Scouting pitching table (`/dashboard/opponents/{id}`) shows `#P` and `Strike %` columns for each pitcher, formatted and placed per Technical Notes.
- [ ] **AC-3**: The Game Box Score pitching lines (`/dashboard/games/{id}`) show `#P` and `Strike %` columns for each pitcher in each team's pitching table, formatted and placed per Technical Notes.

- [ ] **AC-6**: When `pitches` is 0 or NULL for any display element (table row or summary card), Strike % displays as `"-"` (not a division error or blank).
- [ ] **AC-7**: Full test suite (`pytest`) passes with no regressions.
- [ ] **AC-8**: New unit tests call the compute functions directly with crafted input dicts, verifying Strike % is computed correctly (normal case, zero-pitches guard, and NULL-pitches guard).

## Technical Approach
Five display surfaces need the same pattern applied: add columns to SQL SELECT, compute `strike_pct` in the rate-computation function, and add `<th>`/`<td>` to the template. See Technical Notes in the epic for the complete list of files, query locations, and column placement guidance. The opponent scouting query already selects `pitches` but not `total_strikes` -- only the missing column needs to be added there.
