# E-067-01: Add context-layer assessment gate to closure sequence

## Epic
[E-067: Context-Layer Assessment Gate](epic.md)

## Status
`TODO`

## Description
After this story is complete, every epic's closure sequence will include a mandatory context-layer assessment gate that evaluates six triggers, requires explicit per-trigger yes/no verdicts in the epic's History section, and blocks archival until the assessment is complete. When any trigger fires, claude-architect is spawned to codify the findings in the context layer before the epic can be archived.

## Context
E-064 and E-062 both produced durable lessons that were captured only in epic files. When those epics are archived, the lessons become invisible to agents. The existing Documentation Assessment Gate covers user-facing docs but not the context layer (rules, conventions, agent guidance). This story adds a parallel gate for context-layer codification. See epic Technical Notes for CA's full assessment.
