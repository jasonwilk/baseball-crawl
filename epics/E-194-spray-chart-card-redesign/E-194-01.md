# E-194-01: Chart Rendering: figsize, Bunt Marker, Legend

## Epic
[E-194: Spray Chart Card Redesign](epic.md)

## Status

## Description
After this story is complete, spray chart images will render at a compact 3x4 inch size (down from 4x6), bunts will display as distinct down-triangle markers instead of circles, and the legend will include all five contact type categories. These changes improve chart density for the 3-column card layout and give coaches visual distinction for bunters.

## Context
The experiment session confirmed that 4x6 figsize creates excessive whitespace and that bunts being visually identical to ground balls loses coaching-valuable information. The compact size works well in 3-column card layouts for PDF output.

## Acceptance Criteria
- [ ] **AC-1**: Given a spray chart rendered by `render_spray_chart()`, the figure size is 3x4 inches (was 4x6).
- [ ] **AC-2**: Given a spray event with `play_type="bunt"`, the marker shape is a down-triangle (`"v"`), distinct from ground ball circles (`"o"`), per TN-1.
- [ ] **AC-3**: The legend's second row contains five entries in order: Ground Ball, Line Drive, Fly Ball, Popup, Bunt, with `ncols=5`. The Bunt entry uses the `"v"` marker per TN-1.
- [ ] **AC-4**: Given a call to `render_spray_chart(events, title=None)`, no title is drawn on the chart. (Existing behavior -- verify not broken by figsize change.)
- [ ] **AC-5**: Given a call to `render_spray_chart(events, title="Team Spray Chart")`, the title renders correctly at the new figsize.
- [ ] **AC-6**: Existing tests continue to pass.

## Technical Approach
Three changes in `src/charts/spray.py`:
1. The `_PLAY_TYPE_MARKERS` dict maps `"bunt"` to `"v"` instead of `"o"`.
2. `render_spray_chart()` uses `figsize=(3, 4)` instead of `(4, 6)`.
3. `_draw_legend()` adds a fifth handle for Bunt with `"v"` marker and changes `ncols=4` to `ncols=5`.
