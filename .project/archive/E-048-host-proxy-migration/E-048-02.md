# E-048-02: Update addon output paths

## Epic
[E-048: Migrate mitmproxy to Host-Based Standalone Proxy](epic.md)

## Status
`DONE`

## Description
After this story is complete, the mitmproxy addon scripts will write their output to `proxy/data/` instead of `data/mitmproxy/`. Inside the container, these paths are `/app/proxy/data/header-report.json` and `/app/proxy/data/endpoint-log.jsonl`. The `_PROJECT_ROOT` sys.path fallback in `header_capture.py` will also be fixed to use the correct parent index.

## Context
The addons currently write to `/app/data/mitmproxy/` which maps to the project's `data/mitmproxy/` directory. With the host-based proxy, output should go to `proxy/data/` to keep the proxy self-contained. The volume mount `..:/app` means `/app/proxy/data/` maps to `proxy/data/` on the host. Tests mock these path constants, so changing the defaults will not break tests.

## Acceptance Criteria
- [ ] **AC-1**: `proxy/addons/header_capture.py` sets `_REPORT_PATH = Path("/app/proxy/data/header-report.json")`.
- [ ] **AC-2**: The docstring at the top of `header_capture.py` references `proxy/data/header-report.json` instead of `/app/data/mitmproxy/header-report.json`.
- [ ] **AC-3**: `proxy/addons/endpoint_logger.py` sets `LOG_PATH = Path("/app/proxy/data/endpoint-log.jsonl")`.
- [ ] **AC-4**: The `_PROJECT_ROOT` line in `header_capture.py` is changed from `parents[3]` to `parents[2]` so it correctly resolves to `/app` (the project root) inside the container.
- [ ] **AC-5**: All existing tests in `tests/test_proxy/` pass without modification (they mock path constants).

## Technical Approach
1. In `/workspaces/baseball-crawl/proxy/addons/header_capture.py`:
   - Change `_REPORT_PATH = Path("/app/data/mitmproxy/header-report.json")` to `Path("/app/proxy/data/header-report.json")`
   - Change `parents[3]` to `parents[2]` on the `_PROJECT_ROOT` line
   - Update the module docstring path reference
2. In `/workspaces/baseball-crawl/proxy/addons/endpoint_logger.py`:

3. Run `pytest tests/test_proxy/` to confirm no regressions.
