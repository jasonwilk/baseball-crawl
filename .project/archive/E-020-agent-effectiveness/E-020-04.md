4. **"Testing Standards"** (lines 98-100): Duplicates CLAUDE.md's "Testing" section.

Additionally, the memory section is duplicated: "Memory Instructions" (lines 177-189) and "Persistent Agent Memory" (lines 191-213) contain overlapping save/don't-save lists.

Other gaps: no Anti-Patterns section, no Error Handling section.
