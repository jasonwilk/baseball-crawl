# E-191-01: Replace Brand Strings in Templates, Email, and API Config

## Epic
[E-191: Rebrand User-Facing UI from "LSB Baseball" to "Baseball Stats"](epic.md)

## Status
`DONE`

## Description
After this story is complete, the 17 inventoried in-scope items that currently display "LSB Baseball" or "Lincoln Standing Bear" will instead show "Baseball Stats" (or a generic equivalent for the FastAPI description). This covers 17 string replacements across 14 files: auth templates, error templates, base layouts, email content, passkey configuration, and the FastAPI app description.

## Context
The user wants to remove school-identifying information from the public-facing app. A full-codebase inventory identified 58 references across 8 categories; the user chose to change only the 17 items inventoried in categories 1-3 (see Brand Replacement Map in epic Technical Notes). This story covers all 17 in a single pass since they are all independent string replacements with no interdependencies.

## Acceptance Criteria
- [ ] **AC-1**: The login page heading reads "Log in to Baseball Stats" (not "LSB Baseball").
- [ ] **AC-2**: All 7 auth template `<title>` tags use "Baseball Stats" as the brand suffix (login, check_email, verify_error, passkey_register, passkey_prompt, passkey_error, plus the base_auth default title).
- [ ] **AC-3**: The nav header brand text in both `base.html` and `base_auth.html` reads "Baseball Stats".
- [ ] **AC-4**: All 3 error page `<title>` tags (403, 404, 500) use "Baseball Stats" as the brand suffix.
- [ ] **AC-5**: The magic link email subject is "Your login link for Baseball Stats" and the email body says "log in to Baseball Stats".
- [ ] **AC-6**: The passkey `rp_name` in `src/api/routes/auth.py` is "Baseball Stats".
- [ ] **AC-7**: The FastAPI app description in `src/api/main.py` does not contain "Lincoln Standing Bear" or "LSB". It describes the app generically (e.g., "High school baseball coaching analytics platform.").
- [ ] **AC-8**: A grep for `LSB Baseball` across the files listed in Files to Create or Modify returns zero matches.
- [ ] **AC-9**: A grep for `Lincoln Standing Bear` across the files listed in Files to Create or Modify returns zero matches.
- [ ] **AC-10**: All existing tests pass without modification.

## Technical Approach
This is a straightforward find-and-replace. The Brand Replacement Map in the epic's Technical Notes lists all 17 substitutions with exact current strings, replacements, file paths, and line numbers. Items 1-16 are literal `LSB Baseball` -> `Baseball Stats` replacements. Item 17 (FastAPI description) requires a rewrite to remove the school name while preserving the app's purpose description.

The `base.html` default `<title>` (line 6) is already "Baseball Dashboard" and does not need to change -- only the nav brand `<span>` on line 12. Per epic Technical Notes, the "Baseball Dashboard" vs "Baseball Stats" title difference is intentional (function vs brand).
