## Description
After this story is complete, CLAUDE.md contains a concise troubleshooting section that tells agents and the user how to check the app's health, read logs, curl endpoints, and diagnose common issues -- all from inside the devcontainer. This is the "teach a person to fish" story: the networking fix from E-027-01 is useless if nobody knows the commands.

## Context

This story documents the workflow so that Claude Code agents, the user, and any future contributors know exactly how to troubleshoot the running app. The section should be short and practical -- commands, not prose.
