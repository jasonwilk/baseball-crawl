Critically, the current stub references Cloudflare D1 and "wrangler" tooling in the `Database Standards` section. The E-009 decision (finalized 2026-02-28) replaced D1 with SQLite entirely. Any data-engineer dispatched with the current stub may attempt to use `wrangler d1 execute` or similar commands that are now wrong.

The agent-blueprints.md in claude-architect's memory also still describes data-engineer as "to be created when needed" -- this is addressed in E-013-05.

The canonical agent file format is documented in the epic's Technical Notes. The E-009 architecture details (SQLite conventions, ip_outs, soft refs, splits, migration runner) are in the epic's Technical Notes under "E-009 Architecture (for data-engineer)".
