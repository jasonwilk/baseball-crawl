# E-022-03: Add Git Commit Workflow Reminder to CLAUDE.md

## Epic
[E-022: Safety Scan Hardening](./epic.md)

## Status
`DONE`

## Description
After this story is complete, CLAUDE.md's Git Conventions section includes a bullet reminding agents to verify that the PII scan confirmation appears in commit output. This creates structural awareness in every agent that reads CLAUDE.md (which is all of them) without adding enforcement overhead.

## Context
CLAUDE.md is ambient context -- loaded into every agent's system prompt. A single line in the Git Conventions section is the lightest-weight mechanism to create awareness. This is not enforcement (that is what hooks are for); this is a reminder to look for evidence that enforcement ran.

The existing Git Conventions section contains:
```
- Use conventional commits: feat:, fix:, refactor:, test:, docs:, chore:
- Write descriptive commit messages explaining the "why"
- Keep PRs focused on a single concern
- Reference story IDs in commit messages when working on stories
```

Adding one bullet about PII scan verification is proportionate and non-disruptive.

This story has no dependencies and can execute in parallel with E-022-01 and E-022-02.

## Acceptance Criteria
- [ ] **AC-1**: CLAUDE.md Git Conventions section contains a bullet instructing agents to verify PII scan confirmation appears in commit output
- [ ] **AC-2**: The new bullet references the `[pii-scan]` prefix so agents know what to look for
- [ ] **AC-3**: The new bullet tells agents what to do if the confirmation is missing (investigate, do not proceed)
- [ ] **AC-4**: No other CLAUDE.md content is modified -- the change is additive (one bullet added)
- [ ] **AC-5**: The wording is concise (one to two sentences maximum)

## Technical Approach
Add one bullet to the Git Conventions section of `/Users/jason/Documents/code/baseball-crawl/CLAUDE.md`, after the existing bullets:

Read CLAUDE.md first to find the exact location. The Git Conventions section is under `## Git Conventions`. Append the new bullet after the last existing bullet in that section.
